# NLP Project

Este proyecto configura un entorno Jupyter Notebook con bibliotecas orientadas a procesamiento de lenguaje natural (NLP).

## Requisitos

- Docker
- Docker Compose

1. Clona este repositorio:

```bash

```

### Accede a Jupyter Notebook en tu navegador:

```bash

```
